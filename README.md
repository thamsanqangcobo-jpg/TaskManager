Java Task Manager Application
Overview

The Java Task Manager is a desktop application developed using Java Swing in NetBeans. The system allows users to create, manage, and organize tasks through an interactive graphical user interface.

The application demonstrates key Java programming concepts including GUI development, database integration, collections, and file handling.

Features

Add new tasks

Edit existing tasks

Delete tasks

Categorize tasks

Display tasks in a table interface

Store tasks using an SQLite database

Export task data to CSV files

View file properties using Java NIO

Technologies Used

Java

Java Swing (GUI)

SQLite Database

JDBC

Java Collections

Java NIO (File Handling)

NetBeans IDE

System Components
Task Model

Represents the task data structure including:

Task name

Description

Category

Status

Date created

GUI Interface

Built using Java Swing and includes:

JFrame window

JTable for displaying tasks

Buttons for managing tasks

Input fields for task information

Database Integration

The system uses SQLite with JDBC to store and retrieve tasks from a local database.


How to Run the Project

Clone the repository:

Open the project in NetBeans IDE

Ensure the SQLite database file is available

Run the project from NetBeans
